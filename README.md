## 🚀 Featured Projects

### [📊 DataTrader](https://github.com/wwelden/DataTrader)
Full-stack trading journal for stocks and options — user auth, CSV import, position tracking, and analytics. Built server-rendered without client-side JavaScript by using HTMX for interactivity.

![Go](https://img.shields.io/badge/-Go-00ADD8?style=flat-square&logo=go&logoColor=white) ![HTMX](https://img.shields.io/badge/-HTMX-3366CC?style=flat-square&logo=htmx&logoColor=white) ![Templ](https://img.shields.io/badge/-Templ-gray?style=flat-square) ![SQLite](https://img.shields.io/badge/-SQLite-003B57?style=flat-square&logo=sqlite&logoColor=white)

### [🇬🇧 British](https://github.com/wwelden/British)
An esoteric interpreted programming language with British-inspired syntax — every statement ends with `innit`. Full lexer, parser, evaluator, and REPL with OOP support, control structures, and AST transformations.

![Haskell](https://img.shields.io/badge/-Haskell-5D4F85?style=flat-square&logo=haskell&logoColor=white) ![Happy](https://img.shields.io/badge/-Happy-gray?style=flat-square) ![Alex](https://img.shields.io/badge/-Alex-gray?style=flat-square)

### [🐒 Go Interpreter (Monkey)](https://github.com/wwelden/Go-Interpreter)
Interpreter for the Monkey language following Thorsten Ball's *Writing an Interpreter in Go*. Lexer → parser → evaluator, with plans to extend into a full compiler.

![Go](https://img.shields.io/badge/-Go-00ADD8?style=flat-square&logo=go&logoColor=white)

### [⚙️ TIPC Compiler](https://github.com/wwelden/Compilers)
Compiles low-level ILOC intermediate representation into optimized assembly. Implements register allocation and dependency-based instruction scheduling.

![C++](https://img.shields.io/badge/-C++-00599C?style=flat-square&logo=cplusplus&logoColor=white)

### [⌨️ Keystroke](https://github.com/wwelden/Keystroke)
A Markdown-to-HTML compiler with 103 tests across a full lexer → parser → AST → renderer pipeline. Supports math expressions, checklists, and standard Markdown elements.

![TypeScript](https://img.shields.io/badge/-TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white) ![Jest](https://img.shields.io/badge/-Jest-C21325?style=flat-square&logo=jest&logoColor=white)

### [⚔️ Chefs Vs. Aliens](https://github.com/JorgeFLarach/CVA)
Parody tower-defense game in Unity, Plants vs. Zombies-inspired. Served as **lead developer** — designed all battleground mechanics, graphics, and UI, and managed the team's Git workflow.

![C#](https://img.shields.io/badge/-C%23-239120?style=flat-square&logo=c-sharp&logoColor=white) ![Unity](https://img.shields.io/badge/-Unity-000000?style=flat-square&logo=unity&logoColor=white)
